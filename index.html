Übersicht

TachyGIS ist eine Idee zur archäologischen Grabungsvermessung mit Tachymeter und GIS.

QGIS-Plugins erlauben es, Funde und Befunde mit einem Tachymeter "live" einzumessen. Neben einer Tachymeter-Schnittstelle, die die gemessenen Punkte an QGIS übergibt und als Standard-Geodaten speichert, wurde eine archäologiespezifische Benutzeroberfläche entwickelt, die eine effektive Datenerfassung vor Ort ermöglicht.

TachyGIS fasst zwei (ursprünglich) getrennte Entwicklungen zusammen:

  • Tachy2GIS bildet eine QGIS-Schnittstelle zum Tachymeter
  • Tachy2GIS_arch bildet eine archäologische Fachschale für Tachy2GIS

In der neuen TachyGIS Version 2 sind diese Komponenten nun auch softwareseitig zusammengeführt. Neben dem aufwändigen Code-Refactoring werden nun Geopackages eingesetzt (anstatt der bisherigen Shapefiles) und es gab Überarbeitungen (und Fehlerbereinigungen) der Tools und der Nutzeroberfläche.

Version 2 (aktuell)

Die TachyGIS Version 2 ist die aktuelle Version.

Sie ist eine grundlegende Überarbeitung der Version 1. Wesentliche Funktionalität blieb erhalten. Schwerpunkte der Überarbeitung waren:

  • Code-Refactoring (nur noch ein Plugin)
  • Geopackage (anstatt bisher Shapefiles)
  • Korrekturen an Tools und Oberfläche

Weitere Informationen und Downloads findet man auf der folgenden Seite:

  • TachyGIS Version 2

Version 1

Die TachyGIS Version 1 ist der Vorgänger der aktuellen Version 2.

Sie stellt Hilfsmittel (Plugins) für QGIS zur Verfügung, die eine direkte tachymetrische Vermessung realer 3D-Geometrien sowie eine effektive Erfassung archäologischer Attribute für Grabungspläne ermöglichen.

English Version

TachyGIS is an idea to survey archaeological excavations with total station and GIS. With QGIS plugins it is possible to directly use total station 3D survey points to edit points, lines and polygons in QGIS and to efficiently record archaeological attributes. Some more information is available at:

  • TachyGIS – An Idea to Survey Archaeological Excavations with Total Station and GIS

There are some important GitHub-repositories belonging to TachyGIS:

  • Tachy2GIS_arch is an archaeological user interface for Tachy2GIS (by Landesamt für Archäologie Sachsen and Generaldirektion Kulturelles Erbe Rheinland-Pfalz)
    • latest productive version is 2.0: xxxTachy2GIS_arch_v2.0xxx,
    • former productive version is 1.1: xxxTachyGIS_arch_v1.1xxx,
  • Tachy2GIS is a QGIS interface for total station (by Archäologisches Museum Hamburg), that now is completely integrated in version 2 of TachyGIS_Arch.

(This TachyGIS-site tries to summarize information spanning projects around the TachyGIS idea.)
