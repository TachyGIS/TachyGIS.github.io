Übersicht zur Version 1

Die Version 1 von TachyGIS (eigentlich TachyGIS_arch) stellt Hilfsmittel (Plugins) für QGIS zur Verfügung, die eine direkte tachymetrische Vermessung realer 3D-Geometrien sowie eine effektive Erfassung archäologischer Attribute für Grabungspläne ermöglichen:

  • Übernahme von 3D-Koordinaten direkt vom Tachymeter in entsprechende Punkt-, Linien- oder Polygon-Geometrien von QGIS-Layern
  • separater 3D-Viewer für eine angemessene Visulisierung der vermessenen Objekte (inkl. Basis-Editierfunktion in 3D)
  • effektive, auf Grabungsprozesse abgestimmte Erfassung der für einen Grabungsplan erforderlichen Attribute
    • (modifizierbare) Vokabulare für Objekttyp, Datierung und Material sind integriert
  • maßgetreue Transformation aus lokalen Koordinaten in ein ebenes Standard-CRS (z.B. UTM) anhand von Passpunkten, es werden nur Verschiebung und Drehung benutzt (im Vergleich zu einer geodätischen Transformation bleiben die gemessenen Längen und Winkel hier exakt erhalten)
  • diverse Bearbeitungsfunktionen im Kontext eines Grabungsplan
    • ausschneiden, ???, …
  • Profile-Tool: Entzerrung von Profil-Fotos anhand gemessener Passpunkte und Transformation in eine gängige Plandarstellung (am Profil, parallel zum Profil oder zum Plan)
  • speziell konfigurierte Layer ermöglichen  eine standardisierte Plandarstellung mit den Zeichenvorschriften der Vorlage (änderbar)
  • Die gemessenen und weitere benutzte Geodaten werden als Shapefiles gespeichert.
  • Die Arbeit mit TachyGIS erfordert eine spezielle QGIS-Projektvorlage (mitgeliefert).

Die ersten beiden Punkte übernimmt das Plugin „Tachy2GIS“, die weitere Funktionalität entstammt aus dem Plugin „Tachy2GIS_arch“.

Die Software ist frei verfügbar unter Tachy2GIS_arch, eine dazugehörige QGIS-Projektvorlage unter GIS-Projekt_UTM33.

Auf GitHub findet man ein knappes Handbuch zu Tachy2GIS und Tachy2GIS_arch.
